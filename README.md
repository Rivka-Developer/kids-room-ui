# 🎈 Kids Room Decor - Online Store UI

ממשק משתמש (UI) מעוצב עבור חנות מקוונת לעיצוב חדרי ילדים. הפרויקט מתמקד בחוויית משתמש נעימה, צבעונית ומזמינה.

## 🎨 על הפרויקט
זהו פרויקט Frontend המציג קטלוג מוצרים לעיצוב חדרי ילדים. האתר כולל עיצוב רספונסיבי המותאם לכל סוגי המסכים (מחשבים, טאבלטים וטלפונים ניידים).

## ✨ תכונות האתר
* **דף נחיתה מעוצב:** כולל באנר ראשי וניווט נוח.
* **גלריית מוצרים:** תצוגה של פריטי עיצוב עם שמות ומחירים.
* **עיצוב מותאם לילדים:** שימוש בפלטת צבעים רכה וגופנים קריאים.
* **Responsive Design:** האתר נראה מצוין בכל מכשיר.

## 🛠️ טכנולוגיות בשימוש
* **HTML5:** מבנה סמנטי של דפי האתר.
* **CSS3:** עיצוב מתקדם הכולל Flexbox ו-Grid ליצירת פריסה גמישה.
* **Google Fonts:** שימוש בגופנים מותאמים אישית.
