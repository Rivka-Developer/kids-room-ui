# Kids Room Decor - Online Store UI

A professionally designed User Interface (UI) for an online children's interior decor store. The project focuses on creating a pleasant, colorful, and inviting user experience.

## Project Overview

This is a Frontend project showcasing a product catalog for kids' room styling. The website features a fully responsive design, optimized for all screen sizes including desktops, tablets, and mobile devices.

## Key Features

* **Designed Landing Page:** Includes a main hero banner and intuitive navigation.
* **Product Gallery:** Clear display of decor items with names and pricing.
* **Child-Friendly Aesthetic:** Uses a soft color palette and highly readable typography.
* **Responsive Design:** Seamless layout adjustments for any device using modern CSS techniques.

## Technology Stack

* **HTML5:** Semantic structure for web pages.
* **CSS3:** Advanced styling utilizing Flexbox and CSS Grid for flexible layouts.
* **Google Fonts:** Integration of custom web fonts for enhanced typography.
